# User Story: Third-Party API Integrations - External Media Embeds

**As a** Versa User
**I want to** embed external media (e.g., YouTube videos, Spotify playlists) into my content or profile
**So that I can** enrich my shared experience

## Acceptance Criteria:

*   **Given** a Versa user wants to embed external media,
*   **When** the user provides a valid URL from a supported external media service,
*   **Then** the external media is properly embedded and displayed within Versa.
*   **And** the embedded media is playable directly within the Versa application.
