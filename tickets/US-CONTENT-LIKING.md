# User Story: Advanced Content Interaction - Liking

**As a** Versa User
**I want to** 'like' content
**So that I can** express my appreciation and indicate interest

## Acceptance Criteria:

*   **Given** a Versa user is viewing content,
*   **When** the user 'likes' the content,
*   **Then** the 'like' count for the content is updated.
*   **And** the user's 'like' status is recorded.
