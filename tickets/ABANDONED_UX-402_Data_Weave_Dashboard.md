Ticket ID: UX-402
Title: Design the "Data Weave" Dashboard Screen in Google Stitch
Assignee: Green Lantern
Priority: High

Objective:
Create a high-fidelity mockup and interactive prototype for the new "Data Weave" dashboard, a central hub where users can visually map and stitch together different data sources before running transformations.

Key Requirements:
1. Source Node Library Panel: A left-hand sidebar containing draggable, pre-styled components representing various data integrations (APIs, Databases, CSV uploads).
2. Canvas Area (The Loom): A drag-and-drop workspace where users can connect source nodes with visual "threads" (splines) to define data flow.
3. Transformation Configuration Drawer: A slide-out panel that appears when a user clicks on a connection thread, allowing them to define how the data is stitched together (e.g., joins, filters, formatting).
4. Google Material 3 Alignment: The entire interface must strictly adhere to Google's Material 3 design guidelines, ensuring it feels native to the broader Google ecosystem.

Deliverables:
- High-fidelity screens (Desktop & Tablet breakpoints) generated within Google Stitch.
- Interactive click-through prototype demonstrating the drag-and-drop flow and the configuration drawer interaction.
- A library of exported SVG assets and a mini design-spec document detailing spacing, typography, and color tokens used for handoff to the frontend team.

Next Steps:
Green Lantern should review the existing wireframes, gather the necessary Material 3 components, and begin generating the high-fidelity elements in Google Stitch. Once the initial draft is complete, please schedule a quick review session with the product team.
