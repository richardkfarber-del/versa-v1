# Versa Initial User Stories

## As a User (Partner A)...
1.  **Partner Linking:** I want to securely link my account with Partner B using a QR code or invite link so we can share a session without merging our private accounts.
2.  **The Connection Compass:** I want to take a "blind match" desire quiz with Yes/Maybe/No options so I can express my interests without pressure.
3.  **The Match Reveal:** I want to see only the activities where both my partner and I answered "Yes" or "Maybe" so we can focus on shared desires without feeling rejected or exposed.
4.  **Active Date Night:** I want to be guided through a synced 15-minute activity with my partner so we can focus on connection and intimacy.
5.  **The Digital Safe Word:** I want a highly visible "Red Light" or "Pause" button during the activity so I can immediately stop the session if I feel uncomfortable or triggered.
6.  **The Grounding Screen:** I want the screen to instantly shift to a soothing color and display a breathing exercise if either of us presses the "Red Light" button so we can regulate our emotions together safely.
7.  **Privacy:** I want to know my sensitive psychological data and desires are end-to-end encrypted so I can trust the app with my most intimate thoughts.
