# User Story: Admin Dashboard for Content Moderation

**As a** Versa Administrator
**I want to** manage user-generated content and enforce platform guidelines
**So that the platform** remains safe and compliant

## Acceptance Criteria:

*   **Given** a Versa Administrator logs into the admin dashboard,
*   **When** the administrator navigates to the content moderation section,
*   **Then** the administrator can view reported content and user comments.
*   **And** the administrator can approve, reject, or remove content/comments.
*   **And** the administrator can view a history of moderation actions.
